### 添置友链

#### 基本要求
1. 站点可正常访问，HTTPS优先；  
2. 内容健康、无恶意脚本/广告轰炸；  
3. 更新频率 ≥「半年更」即可，不强制活跃；  

#### 技术细节
- 建议先添加本站友链  
- 完成后在下方评论留言或在[Github仓库](https://github.com/afingpo/blog-data)提交一个PR，格式见仓库，本站信息为下：
```json
{
"title": "涵哲子居",
"imgurl": "https://afipo.top/logo.png",
"desc": "天哲地理，共公卿好",
"siteurl": "https://afipo.top",
"feed": "https://afipo.top/rss.xml"
}
```
